# ─── YouTube: cookie export (strongly recommended on datacenter servers) ───────
#   Install "Get cookies.txt LOCALLY" extension in Chrome
#   Visit youtube.com while logged in → Export cookies → save as yt_cookies.txt
#   Set in .env: YOUTUBE_COOKIES_FILE=/path/to/yt_cookies.txt
#
#   Alternative (headless server):
#   yt-dlp --cookies-from-browser chrome --cookies /path/yt_cookies.txt https://youtube.com

# ─── RapidAPI fallback for YouTube (when yt-dlp gets 403) ────────────────────
#   Sign up free at https://rapidapi.com
#   Subscribe to "youtube-search-download3" (boztek-technology) — free tier available
#   Set in .env: RAPIDAPI_KEY=your_key_here

# ─── Instagram cookie export ──────────────────────────────────────────────────
#   Same method as YouTube — export cookies from instagram.com session
#   Set in .env: INSTAGRAM_COOKIES_FILE=/path/to/instagram_cookies.txt

# ─── Spotify credentials (optional, needed for playlist downloads) ────────────
#   Register at https://developer.spotify.com/dashboard → Create App
#   Set in .env: SPOTIFY_CLIENT_ID=... and SPOTIFY_CLIENT_SECRET=...
